## Task Description ##

給你一個正整數 $N$ ($1 < N \le 10^{18}$)，請把 $N$ 質因數分解。 

## Input ##

第一行是一個正整數 $T$ ($1 \le T \le 800$)，表示測資數量。接下來 $T$ 行，每行一個正整數 $N$ ($1<N \le 10^{18}$)。 

## Output ##

對於每個 $N$，輸出 $N$ 的質因數分解。格式參見範例。

## Sample Input ##
```
3
60
36
10007
```

## Sample Output ##
```
60 = 2^2 * 3 * 5
36 = 2^2 * 3^2
10007 = 10007
```
