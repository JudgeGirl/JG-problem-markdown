Task Description

Write a program to read a positive interger $n$ and a sequence of $n$ positive intergers, then print the sequence in $2$ lines. The first line is for odd numbers, and the second line is for even numbers.

Limits

$n$ is no more than $1000$.

Input Format

There are two lines in the input. The first line has the integer $n$. The second line has the $n$ positive inetgers.

Output Format

There are two lines in the output, according to the description.

Sample Input

5
3 6 7 9 20

Sample Output

3 7 9
6 20
